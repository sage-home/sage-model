## Quick Reference

### Essential Commands
```bash
# Build configurations
make physics-free      # Core properties only
make full-physics      # All properties  
make custom-physics CONFIG=file.json

# Testing
make tests            # All tests
make core_tests       # Core infrastructure only
./test_sage.sh        # End-to-end scientific validation

# Property access patterns
GALAXY_PROP_Mvir(galaxy)              # Core properties: direct access
get_float_property(galaxy, prop_id)   # Physics properties: generic access
```

### Key File Locations
- **Properties**: `src/properties.yaml` ‚Üí Auto-generates property system
- **Parameters**: `src/parameters.yaml` ‚Üí Auto-generates parameter system  
- **Config**: `input/config.json` ‚Üí Runtime module configuration
- **Templates**: `docs/templates/` ‚Üí Code templates for development

### Critical Architecture Principles
1. **Core-Physics Separation**: Core infrastructure is physics-agnostic
2. **Property Build System**: Build-time optimization of property sets
3. **Module Registration**: Self-registering modules via C constructors
4. **Pipeline Phases**: HALO ‚Üí GALAXY ‚Üí POST ‚Üí FINAL execution
